Explanation of C Code - display Function

  1. Clearing the screen and declaring variables.
  2. Opening the "username.txt" file in binary mode for reading.
  3. Declaration of a structure variable u1.
  4. Checking if the file pointer fp is NULL. If NULL, prints an error message.
  5. Looping through the file to read user records.
  6. Checking if the username1 matches the username in the record. If matched, displays user account information.
  7. Printing user account information including name, father's name, mother's name, Aadhar card number, mobile number, date of birth, address, and account type.
  8. Closing the file.
  9. Displaying a menu for different actions a user can perform.
  10. Prompting the user to enter their choice.
  11. Using a switch statement to perform actions based on user input:
    • 1: Calls the checkbalance function.
    • 2: Calls the transfermoney function.
    • 3: Calls the logout function and then the login function.
    • 4: Exits the program.
